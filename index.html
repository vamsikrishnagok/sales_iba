Displays Webex Assistant's real-time transcription inside the app.

1. Sign in to Webex

Create an Integration (not a personal token) at developer.webex.com → My Webex Apps → Create an Integration. The meetings SDK needs a downscopable OAuth token, which only an Integration can issue. Paste this exact value into your Integration's Redirect URI(s) field, and add the scopes spark:all and spark:kms.

Sign-in: Not signed in

Advanced: use a personal access token instead Personal tokens work for the REST check but usually fail the meetings auth (they cannot be downscoped). Prefer the Integration login above.
If this app is running inside a Webex meeting, the meeting is auto-detected. Otherwise, paste the meeting destination (SIP URI, meeting number, or link).
