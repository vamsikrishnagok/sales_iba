Live Transcript → FastAPI

Streams Webex Assistant's real-time transcription to your backend.

1. Configuration

Endpoint receiving POST JSON payloads. Must be HTTPS when this page is served via HTTPS (GitHub Pages). Required to authenticate the Webex JS SDK. The token is held in memory only and never transmitted anywhere except to Webex. The Webex JS SDK needs to join the meeting as a participant in order to receive transcriptions. Webex Assistant must already be enabled for the meeting.
